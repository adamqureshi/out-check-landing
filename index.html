FAQ

The trust-building answers should be one scroll away.

Do I have to share my Tesla password?

No. Sellers connect via Tesla’s sign-in flow (OAuth). You should also explain exactly what permissions you request and how users can revoke access.

Is the connection read-only?

It should be. If you request any non-read scope, explain why—in plain English—and offer an alternative if possible.

How long does it take?

Most users should finish in about 2 minutes: connect → select vehicle → generate report.

What exactly is “battery health”?

Be precise. If you provide an estimate (not a service-mode measurement), say so and explain the method. Avoid vague claims.

Are you affiliated with Tesla?

No. Add a clear, consistent disclaimer in the footer and on your connect screen.
